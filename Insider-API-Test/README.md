# Insider API Test

RestAssured ile Petstore API test otomasyonu.

## Çalıştırma

```bash
mvn clean test
```

## Test Senaryoları

- Store Inventory API kontrolü
- Store Order CRUD operasyonları

## Raporlar

- `target/extent-reports/` - Extent raporları
- `target/surefire-reports/` - TestNG raporları
